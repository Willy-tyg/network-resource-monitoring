pour pouvoir lancer l'agent veillez executé ces quelques étapes:

* Nous supposons que vous disposez deux fichiers: le script  script.sh et le fichier client

1- rendez vous dans le dossier contenant ces deux fichiers
2- donnez les droits d'exécution au script : "sudo chmod 770 script.sh" 
2- executez le script en tapant la commande "sudo ./script.sh"
3- verifiez que le service à bien été crée et est foncionnel en tapant la commande "systemctl status system-info-collector.service"
